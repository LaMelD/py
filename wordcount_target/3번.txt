"프로그래밍 언어"

C, C++를 주력언어로 사용하고 있고, 서브 언어로는 Java, Python을 사용하고 있습니다. 현재는 Python을 위주로 사용하고 있습니다. 잘 알려져 있는 queue, stack, single linked list, double linked list, binary tree 등을 구현할 수 있으며 라이브러리를 사용할 수 있습니다. 알고리즘에 대한 학습을 통해 시작복잡도를 계산할 수 있으며 시간복잡도에 따른 코딩이 가능합니다.

"프로젝트"

1. 인테리어 헬퍼

셀프 인테리어를 도와줄 수 있는 인테리어 헬퍼 프로젝트를 진행한 경험이 있습니다. 사용자가 방의 사이즈를 입력하여 방의 모델을 만들고 그 내부를 커스텀하고 그 결과를 3D로 확인할 수 있으며 사용자들끼리의 인테리어를 공유할 수 있게하는 것이 목표인 프로젝트였습니다. Unity3D를 사용하여 구현하였으며, 사용자를 구분하기위해 중복을 허용하지 않는 ID를 입력하도록 하였습니다. 사용자들의 방 인테리어를 공유할 수 있도록 업로드와 뷰를 생성하여 제공하였습니다. API 서버로는 Spring을 사용하였습니다. 사용자 정보와 인테리어를 저장하기 위해서 mysql 서버를 사용하였습니다. 프로젝트의 버전관리는 git과 github를 사용하여 관리하였습니다.

2. 특가 상품과 트랜드

openstack을 사용하여 프라이빗 클라우드를 사용하여 사내를 운영하고 있는 회사에서 사용시간이 적은 시간에 남는 자원과 퍼블릭 클라우드를 사용하여 쇼핑몰에서 특가 상품을 크롤링하여 각 사이트의 특가상품을 리다이렉팅하며, 특가상품에서의 단어의 빈도를 찾아 트랜드를 분석하여 보여주는 것이 목표인 프로젝트였습니다. 이 프로젝트에서 제가 맡은 부분은 크롤링 코드 작성, Hadoop MapReduce 코드의 작성, WEB/WAS 구현 및 DB table 작성이였습니다. Python으로 크롤링코드를 작성하고 각 함수를 모듈화시키고 로직을 구현하였으며, Java를 사용하여 Mapreduce 코드를 작성하였으며, WEB은 아파치를 사용하여 하여 HTML, CSS, Javascript를 사용하여 정적 웹사이트로 작성하였으며, WAS는 Spring을 사용하여 db에 접근하여 정보를 사용하는 형식으로 사용하였습니다. DB는 mysql 서버를 구축하여 서비스로 선택된 특가 상품 관련 테이블과 트랜드를 나타낼 테이블로 구성하였습니다. 각각 서버의 운영체제로는 데비안 계열의 Ubuntu, 레드햇 계열의 Centos를 혼용하여 사용하였습니다.

"도커, 쿠버네티스"

도커와 쿠버네티스를 학습하며 docker의 Dockerfile의 문법을 익히고, 컨테이너의 짜임을 어떻게 해야하는지를 학습하였습니다. 또한 docker-compose 및 docker swarm에 대한 전반적인 내용에 대해 학습하였으며, docker in docker를 사용하여 클러스터를 구축하여 WEB, WAS, DB 서버를 각각 생성하여 게시판 어플리케이션을 만들고 인그레스를 사용하여 배포해본 경험이 있습니다. 쿠버네티스를 학습하여 docker swarm에서 구축했던 게시판 어플리케이션을 기반으로하여 GKE에 구축하여 배포해본 경험이 있습니다. 이후에는 django를 사용하여 web과 was를 구축하고 DB 서버를 두어 같은 게시판 어플리케이션을 GKE에 구축한 경험이 있습니다. 또한 가상머신을 이용하여 kubespray로 클러스터를 구축한 경험이 있습니다.
